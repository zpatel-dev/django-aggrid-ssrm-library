django-aggrid-ssrm — demo

120 years of Olympic history — 271,116 athlete-event records, Athens 1896 to Rio 2016 — served via AG Grid Server-Side Row Model. Dataset from TidyTuesday 2024-08-06, originally compiled by Randi H. Griffin on Kaggle. Try filtering by medal, sorting by year, or dragging sport / noc into the Row Groups panel.
License notice: AG Grid SSRM is an AG Grid Enterprise feature. Without a license key the grid runs in trial mode and shows a watermark. Set AG_GRID_LICENSE_KEY in your env to remove it. This Django library itself is MIT-licensed and independent of AG Grid Ltd.
